# Tower Stack 🏗️

A Tower Bloxx-inspired stacking game — tap to drop blocks and build the tallest tower!

**Play now:** https://nkyang10.github.io/tower-stack/

## How to Play

1. A block swings left and right above the tower
2. **Tap / click** to drop the block
3. The overlapping part stays — the rest gets cut off
4. Keep stacking to grow your tower!
5. If the block misses completely — **game over**
6. Try again to beat your high score!

The tower gets narrower with each imperfect drop, and the swing speed increases as you go higher. Can you get a perfect streak? 🔥

## Features

- 🎮 One-tap gameplay (mouse, touch, or keyboard Space/Enter)
- 📱 Responsive — works on mobile and desktop
- 🌙 Night city aesthetic with animated stars and skyline
- 🎨 Colorful 3D-styled blocks with visual feedback
- 💥 Particle effects and screen shake on landing
- ⭐ "PERFECT!" bonus for spot-on placement
- 🏆 Highscore saved in your browser

## Tech

Built with vanilla HTML, CSS, and Canvas — zero dependencies. Just open `index.html` in any browser.

## Local Development

python3 -m http.server 8080
